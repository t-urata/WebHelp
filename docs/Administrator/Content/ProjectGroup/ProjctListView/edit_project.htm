プロジェクトを編集する

プロジェクトの編集方法について説明します。

 

プロジェクトの編集方法について

  1. [終了]のチェックボックスにチェックを入れると、そのプロジェクトが終了状態になります。

  2. [無効]のチェックボックスにチェックを入れると、そのプロジェクトは無効になります。

  3. [チェックアウトの強制解除]をクリックすると[チェックアウトの強制解除]ダイアログを表示して、任意のプロジェクトのチェックアウト状態を解除できます。

  4. ユーザが編集中のプロジェクトのチェックアウトを強制解除すると、編集中のユーザは編集内容を保存できません。

  5. プロジェクトを無効とすることで、Administrator以外の機能にて管理できなくなります。

  6. 実績工数データが入力されているプロジェクトを無効にすることはできません。無効にしようとした場合、エラーダイアログが開きます。

  7. 作成したプロジェクトは、[プロジェクトの削除]にて削除できます。一度削除したプロジェクトのデータは復元することができません。

  8. 実績工数データが入力されているプロジェクトを無効にする場合は、[実績のみ削除]でプロジェクトの実績データを削除してください。
