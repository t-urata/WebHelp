キーワードフィルタを使う

任意のキーワードを使用して、表示する行を絞り込みます。

 

キーワードフィルタを利用するには

  1. [キーワードフィルタ]の欄に検索するキーワードを入力します。

  2. Enterキーを押します。

  3. キーワードを含む行を表示します。

  4. 複数のキーワードを設定する場合は、間に空白を入力してください。

  5. 複数のキーワードを設定すると、入力した全てのキーワードを含む行を表示します。

 

キーワードフィルタを解除するには

キーワードフィルタを解除する方法は2通りです。

  • [キーワードフィルタ]の欄のをクリックします。

    • は、[キーワードフィルタ]の欄にキーワードを入力すると表示します。

  • [キーワードフィルタ]の欄に入力したキーワードを削除し、Enterキーを押します。
